## 💡 特性

### 桌面版 (≥768px)
```
┌─────────────────────────────────────┐
│         後台左側邊欄 (固定)          │
├─────────┬───────────────────────────┤
│         │                           │
│ 導航    │      主要內容             │

│ (固定)  │   (可滾動)               │
│         │                           │
└─────────┴───────────────────────────┘
```

### 手機版 (<768px)
```
頂部導航欄 [Menu 按鈕]
  ↓ 點擊
滑出側菜單 (半透明遮罩)
  ↓ 選擇項目
導航跳轉 + 菜單關閉
```
