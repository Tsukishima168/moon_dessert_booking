## 二、AI SEO 要做什麼

### 2.1 目前已有的 SEO

- **Metadata：** title、description、keywords（已含台南甜點、安南區美食、療癒甜點等）。
- **結構化資料：** JSON-LD LocalBusiness（名稱、地址、電話、安南區）。
- **頁面策略：** 首頁可索引；結帳、後台 noindex（見 SEO_PAGES_GUIDE.md）。

### 2.2 用 AI 優化的項目（依市場分析結果填入）

| 項目 | 作法 | 備註 |
|------|------|------|
| **Title / Description** | 用「安南區關鍵字 + 品牌 + 行動呼籲」請 AI 產 2–3 組，選一組或微調 | 維持 60 字內 title、150 字內 description |
| **Keywords** | 依安南區關鍵字清單補齊 5–10 個，給 AI 檢查是否自然融入 description | 避免塞關鍵字 |
| **在地文案** | Hero、Footer、關於我們：用「安南區 + 療癒 + 預訂」請 AI 寫短文案 | 保持品牌語氣 |
| **JSON-LD** | 補齊完整地址（本原街一段97巷）、營業時間、服務範圍（安南區果菜市場・本原街／台南／宅配） | 有利在地搜尋與知識面板 |
| **FAQ / 在地頁** | 若有「本原街取貨」「果菜市場附近取貨」「宅配範圍」「預訂方式」等，可請 AI 產 FAQ 結構或一頁在地說明 | 未來可加 FAQ schema |

### 2.3 建議執行順序

1. **完成安南區（果菜市場・本原街）關鍵字與受眾整理** → 產出 `安南區SEO關鍵字.csv` 與 1 頁「受眾＋差異化」摘要。
2. **用 AI 產出候選文案** → 依清單與摘要請 AI 產 title/description/在地短文案 2–3 組（可強調本原街、果菜市場周邊）。
3. **更新專案** → 在 `app/layout.tsx`（或 siteConfig）更新 title、description、keywords；必要時補 JSON-LD 與在地區塊文案。
4. **提交與觀察** → Google Search Console 提交網址、檢查索引；之後用 GA4 看「安南區」「本原街」「果菜市場」相關查詢與流量。
