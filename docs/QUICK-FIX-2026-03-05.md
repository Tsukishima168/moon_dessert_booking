#### 步驟 2: 檢查 Supabase 連接
```javascript
// 在 Console 中執行:
fetch('/api/check-capacity?date=2026-03-05&method=pickup')
  .then(r => r.json())
  .then(console.log)
// 應該返回日期可用性信息
```

#### 步驟 3: 檢查表單初始化
