## 🎯 開發優先級（4 階段）

### 🔴 Phase 1：基礎後台（1-2 個月）
**目標：** 讓店家能管理訂單與商品

1. **登入系統 + 儀表板**
   - 帳號密碼登入
   - Dashboard（今日訂單、營業額、待處理提醒）

2. **訂單管理**
   - 訂單列表（篩選、搜尋、排序）
   - 訂單詳情（完整資訊、狀態變更）
   - 訂單狀態管理（待確認→已確認→準備中→已完成）

3. **商品管理**
   - 商品列表（上架/下架、快速編輯）
   - 商品編輯（名稱、描述、圖片、規格、價格、庫存）
   - 批次操作（批次上架、批次調整價格）

4. **庫存管理**
   - 庫存監控（即時數量、低庫存提醒）
   - 庫存調整（手動調整、進貨記錄）

---

### 🟡 Phase 2：進階功能（2-3 個月）
**目標：** 提升管理效率，增加數據分析

1. **顧客管理**
   - 顧客列表（資料、訂單歷史、累積消費）
   - 顧客分級（VIP、一般會員）
   - 顧客標籤（自訂標籤、標籤篩選）

2. **優惠碼進階**
   - 優惠碼管理後台（使用次數統計）
   - 複雜促銷規則（買 X 送 Y、組合優惠、滿額贈）

3. **營業設定進階**
   - 視覺化日曆設定公休日
   - 時段接單上限（上午/下午分別設定）

4. **基礎報表**
   - 營業額報表（日/週/月/年）
   - 商品銷售排行

---

### 🟢 Phase 3：數據分析（3-6 個月）
**目標：** 完整數據分析與自動化

1. **完整報表系統**
   - 銷售報表（營業額趨勢、商品分析、訂單分析）
   - 顧客分析（回購率、客群分析、LTV）
   - 行銷效果分析（優惠碼 ROI、MBTI 轉換率）
   - 報表匯出（Excel/CSV）

2. **自動化流程**
   - 訂單自動化（確認後自動發送 Email）
   - 庫存自動化（低庫存自動通知、預購自動上架）
   - 顧客自動化（生日月自動發送優惠碼）

3. **第三方整合**
   - LINE 官方帳號（訂單查詢機器人）
   - 金流整合（如果需要）
   - 物流整合（黑貓、超商取貨）

---

### 🔵 Phase 4：進階功能（6 個月以上）
**目標：** 會員系統與進階行銷

1. **會員系統**
   - 會員註冊（註冊送優惠碼）
   - 積分系統（消費累積、積分兌換）
   - 購物金（購物金發放與使用）

2. **進階行銷**
   - 限時優惠（倒數計時器）
   - 滿額贈（自動加贈商品）
   - 組合套餐（套餐商品設定）

3. **行動 App**
   - 管理 App（訂單通知、快速確認）
   - 顧客 App（選配）

## 🏗️ 技術架構建議

### 後台技術選型

**推薦：Next.js Admin Dashboard**
- 與現有前端技術棧一致
- 可共用 Supabase 資料庫
- 快速開發

**技術棧：**
- Next.js 14+ (App Router)
- React Admin / Refine / Shadcn UI
- Supabase Auth（登入系統）
- Recharts（圖表）

### 新增資料表

```sql
-- 管理員帳號
admins (id, email, password_hash, role)

-- 訂單狀態變更記錄
order_status_history (id, order_id, old_status, new_status, changed_by, changed_at)

-- 庫存異動記錄
inventory_logs (id, menu_item_id, change_type, quantity, reason, created_at)

-- 顧客標籤
customer_tags (id, customer_phone, tag_name, created_at)
```
