### 第二階段：資料庫確認（約 15 分鐘）

到 Supabase Dashboard 確認：

| 順序 | 項目 | 操作 | 備註 |
|-----|------|------|------|
| 1 | 訂單表 | Table Editor → orders | 確認欄位完整（含 delivery_*） |
| 2 | 優惠碼表 | Table Editor → promo_codes | 至少有一個有效優惠碼 |
| 3 | 營業設定 | Table Editor → business_settings | 公休、不宅配、接單上限已設 |
| 4 | SQL 腳本 | 若有缺失 | 執行 DELIVERY_SETUP.sql、BUSINESS_SETTINGS_SETUP.sql |
