## 🏗️ 技術架構建議

### 後台技術選型

#### 方案 A：Next.js Admin Dashboard（推薦）
- **優點：**
  - 與現有前端技術棧一致
  - 可共用 Supabase 資料庫
  - 快速開發

- **技術棧：**
  - Next.js 14+ (App Router)
  - React Admin / Refine / Shadcn UI
  - Supabase Auth（登入系統）
  - Recharts（圖表）

#### 方案 B：獨立後台系統
- **優點：**
  - 前後台分離，安全性更高
  - 可獨立部署

- **技術棧：**
  - Next.js Admin 或
  - React + Vite + React Admin

### 資料庫擴充

#### 新增資料表
```sql
-- 管理員帳號
CREATE TABLE admins (
  id UUID PRIMARY KEY,
  email TEXT UNIQUE,
  password_hash TEXT,
  role TEXT, -- 'admin' | 'staff'
  created_at TIMESTAMP
);

-- 訂單狀態變更記錄
CREATE TABLE order_status_history (
  id UUID PRIMARY KEY,
  order_id TEXT,
  old_status TEXT,
  new_status TEXT,
  changed_by TEXT,
  changed_at TIMESTAMP
);

-- 庫存異動記錄
CREATE TABLE inventory_logs (
  id UUID PRIMARY KEY,
  menu_item_id TEXT,
  change_type TEXT, -- 'in' | 'out' | 'adjust'
  quantity INTEGER,
  reason TEXT,
  created_at TIMESTAMP
);

-- 顧客標籤
CREATE TABLE customer_tags (
  id UUID PRIMARY KEY,
  customer_phone TEXT,
  tag_name TEXT,
  created_at TIMESTAMP
);
```
