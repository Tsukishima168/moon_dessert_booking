# 與 moon_map_original 共用圖床說明

Dessert-Booking 與 **moon_map_original** 使用**同一個 Supabase 圖床**顯示產品／選單圖片。

---

## 共用方式

| 項目 | 說明 |
|------|------|
| **Storage** | Supabase bucket：`moon-island-assets` |
| **路徑** | 產品圖：`menu-items/`（檔名為英文代號，如 `classic_tiramisu.webp`） |
| **資料表** | 兩專案皆讀取 **`menu_items`** |
| **欄位** | moon_map_original 使用 `image`；Dessert-Booking 使用 `image_url`，若無則 fallback 到 `image` |

因此，只要**在同一份 Supabase 專案**內更新 `menu_items.image`（或一併更新 `image_url`），兩個網站都會顯示同一批圖片。

---

## 你要做的事

1. **上傳與更新流程**：以 **moon_map_original** 為主  
   - 照片放到 `~/moon-island-upload/menu-items/`，檔名用英文代號（見 `產品品項與檔案名稱對照表.csv`）  
   - 執行 `node upload-to-supabase.js` 上傳  
   - 執行 `node scripts/generate-menu-image-updates.js` 產生 SQL，再到 Supabase SQL Editor 執行  

2. **Dessert-Booking**：無需重複上傳或改程式，重新整理前台即可看到新圖。
