# 前端 UI/UX 優化說明

## 已完成的調整

### 1. 中文介面

- **載入/錯誤**：LOADING → 載入中、ERROR → 無法載入、RELOAD → 重新載入
- **購物車**：YOUR CART → 購物車、ITEMS → 件商品、加入購物車、前往結帳
- **結帳**：CHECKOUT → 填寫訂單、ORDER SUMMARY → 訂單摘要
- **表單欄位**：NAME → 姓名、PHONE → 電話、EMAIL → 電子信箱
- **取貨**：PICKUP DATE → 取貨日期、DELIVERY METHOD → 取貨方式
- **商品**：SOLD OUT → 已售完、ITEMS → 款

### 2. 不擅長網路者的引導

- **三步驟提示**：選甜點 → 加入購物車 → 填資料
- **結帳步驟條**：① 選商品 ② 填資料 ③ 確認訂單
- **欄位 placeholder**：例：王小明、0912-345-678、example@gmail.com
- **填寫說明**：如「訂單確認與匯款資訊將寄至此信箱」

### 3. Banner 展示

- **預覽模式**：後台無 Banner 時會顯示示範橫幅
- **後台 Banner**：在 Supabase 建立 Banner 後會優先顯示
- **新增 Banner**：可執行 `scripts/SEED_DEMO_BANNER.sql` 在資料庫新增示範資料

### 4. 選購與推薦流程

- **本季精選**：無 MBTI 時顯示前 4 款可預訂熱門商品
- **MBTI 推薦**：有測驗結果時顯示為您推薦區塊
- **引導流程**：從「不確定選什麼？試試人氣預訂」進入
