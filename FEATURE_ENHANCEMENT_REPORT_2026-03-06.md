  - 顧客需要提前多少小時預訂
  - 支援自訂

  - 添加商品特殊備註
  - 內部使用說明

- ✅ 搜尋功能
  - 快速查找商品
  - 即時篩選

- ✅ 啟用/停用商品
  - 暫停銷售功能
  - 一鍵停售

**位置:** `/app/admin/menu-availability/page.tsx` (442 行)

**使用方式:**
```
1. 進入後台 → 菜單可用性
2. 搜尋要設置的商品
3. 點擊 [編輯] 按鈕
4. 配置:
   - 周期日期 (選擇可用日期)
   - 時間範圍 (開放/關閉時間)
   - 黑名單日期 (不可用日期)
   - 提前預訂時數
   - 備註說明
5. 保存設置
```

**配置示例:**
```
商品: 漫 claw系列蛋糕
周期: 週一到週五 (工作日)
時間: 09:00 - 20:00
提前預訂: 24 小時
黑名單: 2026-03-08 (公假)
備註: 週末預訂可能延遲
```
